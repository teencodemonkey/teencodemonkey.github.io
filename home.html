Check out my URL Shortener

I created a single page MEAN Stack application that shortens urls. It is similar to the URL shortening services like goo.gl and bit.ly.As of now it has a really long url as it is hosted on Heroku. The long URL of the site kinda destroys the idea of a URL shortener. Nonetheless, it was a very fun beginner project. Check it out at https://nano-it.herokuapp.com/.

How:

This MEAN Stack Single Page Application takes a long url and assigns it a unique ID. Then using that ID, it creates a "short" URL linked to the original long URL. To learn more, visit This link and This one.

Check out my simple Todo App

I created a very simple MEAN Stack todo application. It has very simple user authentication and it uses a basic Regex to verify email accounts. After you create an account, you go to the profile page, and from there you can go to your todo list. The todo list is very simple, and it includes adding a todo, checking it off if it is complete, and a button to clear all the checked off todos. It was a really easy and fun project. Check it out at https://yash-todo-app.herokuapp.com/. You can also find its source code on Github.
