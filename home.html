Email Verification Regex


function validateEmail(email) {
    var re = /\S+@\S+\.\S+/;
    return re.test(email);
};
//validateEmail('1234@abcef.com'); returns true
//validateEmail('hello'); returns false

This is a simple Javascript Regex to verify if a string matches the normal email format (Ex. 1234@abcdef.com).

How:

Regexes are ways to check if a string matches a set format. At first they are pretty confusing and weird, but you get used to them pretty easily. This regex checks if a string has the format of **@**.** where ** represents any form of text.
